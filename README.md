# 無人機地面控制系統 操作手冊

本系統以 PlayStation 手把透過藍牙控制無人機飛行，並搭載夾爪機構進行空中作業。

---

## 系統架構

```
[PS 手把] --藍牙--> [電腦 controller.py] --藍牙(COM4)--> [Arduino] --IBUS--> [飛控 F405]
                                                                    |
                                                                    +--PWM--> [夾爪伺服馬達 D6]
```

| 元件 | 說明 |
|------|------|
| PS 手把 | 透過藍牙與電腦配對，作為輸入裝置 |
| 電腦 (`controller.py`) | 讀取手把輸入，打包成 7-byte 封包透過藍牙串列傳給 Arduino |
| Arduino (`sketch_mar26a.ino`) | 接收指令，轉換為 IBUS 協議送給飛控；同時直接驅動夾爪 |
| 飛控 F405 | 接收 IBUS 訊號控制四軸馬達飛行 |

---

## 硬體需求

- PlayStation 手把（藍牙）
- Arduino（藍牙模組接 D10 RX / D11 TX）
- 夾爪伺服馬達（接 Arduino D6）
- F405 飛控板
- Windows 電腦（已安裝 Python 3）

---

## 環境安裝

```bash
pip install pygame pyserial
```

將 `sketch_mar26a.ino` 燒錄到 Arduino。

## 啟動前設定

開啟 `controller.py`，確認以下參數與實際硬體一致：

```python
COM_PORT = 'COM4'   # Arduino 藍牙模組的 COM 埠號
BAUD_RATE = 9600
```

> 若不確定埠號，請至裝置管理員查詢「藍牙串列連接埠」。

## 啟動方式

```bash
python controller.py
```

## 控制按鍵說明

### 飛行控制（搖桿）

| 搖桿 | 軸向 | 功能 |
|------|------|------|
| 左搖桿 上/下 | 油門 Throttle | 上推加速、下拉減速 |
| 左搖桿 左/右 | 偏航 Yaw | 原地左轉 / 右轉 |
| 右搖桿 上/下 | 俯仰 Pitch | 前傾飛行 / 後傾飛行 |
| 右搖桿 左/右 | 翻滾 Roll | 左傾飛行 / 右傾飛行 |

### 飛行控制（按鍵）

| 按鍵 | 功能 |
|------|------|
| 方向鍵 上 / 下 | 調高 / 調低「基準油門」（巡航高度） |
| 方向鍵 右 / 左 | 增大 / 縮小每次調整的油門步進量（1～20） |
| **□（Square）** | 搖桿校準歸零（搖桿放置中間後按） |
| **△（Triangle）** | **解鎖 / 上鎖**（油門須歸零才能解鎖） |
| **Options** | 長按 **3 秒**安全關閉程式 |

### 夾爪控制

| 按鍵 | 功能 |
|------|------|
| **L1** | 夾爪閉合 |
| **R1** | 夾爪張開 |

## 終端機狀態顯示說明

```
狀態:解鎖 | 總油門:120 | 夾爪:200 | Y:128 P:140 R:115
```

| 欄位 | 說明 | 範圍 |
|------|------|------|
| 狀態 | 解鎖 / 上鎖 | — |
| 總油門 | 當前實際輸出油門 | 0 ~ 255 |
| 夾爪 | 夾爪開合程度 | 0（全閉）~ 255（全開）|
| Y / P / R | 偏航 / 俯仰 / 翻滾 | 0 ~ 255（128 為中立）|

## 標準飛行流程

1. 確認 Arduino 已上電並與電腦藍牙配對
2. PS 手把透過藍牙與電腦配對
3. 執行 `python controller.py`
4. 確認終端機顯示連線成功
5. **搖桿置中**，按下 **□** 進行校準
6. 確認油門歸零後，按 **△** 解鎖
7. 慢慢推左搖桿上方起飛
8. 降落並油門歸零後，按 **△** 上鎖
9. 長按 **Options 3 秒**安全關閉程式

> ⚠️ **安全提醒**：解鎖前必須確保油門值 ≤ 5，否則無法解鎖。

## 偵錯工具

### 測試手把按鍵編號

若更換不同品牌手把，可先用此工具確認按鍵編號：

```bash
python test_joy.py
```

### 監看 IBUS 封包

確認 Arduino 是否正確產生 IBUS 訊號（需將 Arduino USB 接到電腦，COM3）：

```bash
python test_ibus.py
```

正常輸出範例：
```
📦 成功攔截 IBUS! | 翻滾:1500 | 俯仰:1500 | 油門:1000 | 偏航:1500 | 夾爪:1000
```

## 進階參數調整

開啟 `controller.py` 上方的調音台區塊：

```python
JOYSTICK_SENSITIVITY = 60   # 搖桿推滿時的額外油門推力
TILT_SENS = 0.6             # 俯仰 / 翻滾靈敏度（0.0 ~ 1.0）
YAW_SENS  = 0.5             # 偏航旋轉靈敏度（0.0 ~ 1.0）
STEP_SPEED = 5              # 夾爪開合速度
```

## 通訊協議說明（開發參考）

### 電腦 → Arduino（藍牙，9600 baud）

每 40ms 發送一次，共 7 bytes：

```
[ 'S' | Throttle | Yaw | Pitch | Roll | Gripper | Arm ]
```

所有數值範圍皆為 0 ~ 255。Arm 解鎖為 255，上鎖為 0。

### Arduino → 飛控（IBUS，115200 baud）

標準 32-byte IBUS 封包，每 20ms 發送一次：

| 通道 | 功能 | 範圍 |
|------|------|------|
| CH1 | 翻滾 Roll | 1000 ~ 2000 |
| CH2 | 俯仰 Pitch | 1000 ~ 2000 |
| CH3 | 油門 Throttle | 1000 ~ 2000 |
| CH4 | 偏航 Yaw | 1000 ~ 2000 |
| CH5 | 解鎖 Arm | 1000 / 2000 |
| CH6 | 飛行模式（固定自穩）| 2000 |
